opendata.fit

Part of the ACCS project funded by the ARDC. Supported by UNSW.

A data analysis and publication platform for the web

opendata.fit is a cloud-based web platform and execution engine made for managing scientific analysis workflows and datasets from development to publication.

  • Upload instrument data
  • Develop and execute analysis algorithms in the cloud
  • Draw from and contribute to a library of existing open source analysis algorithms
  • Visualise results
  • Publish complete, citable workflows

Enabling reproducible science, from data collection to publication

A complete, portable and citable record of your datasets, analysis process and results

Fully containerised and executable on any machine

Built on FAIR data principles, opendata.fit packages all aspects of your scientific workflow into a single portable JSON format. This file is a complete, portable and citable record of your input datasets, analysis process and results. Fully containerised, your workflow can be executed on any machine.
