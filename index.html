A data analysis and publication platform for the web

opendata.fit is a web platform and cloud-based execution engine made for managing scientific analysis workflows and datasets from development to publication, with a focus on model fitting and optimisation.

Analysis and publication simplified

Easy-to-use web-based user interface optimised for model fitting. Every step of the analysis process is recorded in history.

Publish complete, reproducible workflows by citing a single link.

A complete, portable and citable record of your datasets, analysis process and results

Built on FAIR data principles, opendata.fit packages all aspects of your scientific workflow into a simple JSON file format called a datapackage, based on the open source Frictionless Data Standard. This file is a complete, portable and citable record of your input datasets, analysis process and results.

Analysis algorithms are fully containerised through Docker maintaining a consistent execution environment.

Embed facets of your workflows and data in any webpage with embeddable widgets
