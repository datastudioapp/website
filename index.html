A data analysis and publication platform for the web

opendata.fit is a cloud-based web platform and execution engine made for managing scientific analysis workflows and datasets from development to publication, with a focus on model fitting and optimisation.

Analysis, simplified

Simple, easy-to-use web-based user interface optimised for model fitting. Every step of your analysis process is recorded in history.

Publish complete workflows by citing a single link.

A complete, portable and citable record of your datasets, analysis process and results

Built on FAIR data principles, opendata.fit packages all aspects of your scientific workflow into a single, portable JSON format. This file is a complete, portable and citable record of your input datasets, analysis process and results.

Fully containerised, your packaged workflow can be executed on any machine.

Embeddable data visualisations

opendata.fit provides embeddable widgets to visualise your data on any webpage.

Built on open source data standards

Built on and extending the open source Frictionless Data standard and Vega Visualisation language.
